Как и было указано в требованиях, в папке configs лежит 4 конфигурации менеджера:
    1)Полностью из своих джарников из папки artifacts
    2)Полностью из сових джарников, кроме Readera, который взят из libs\\timofeev
    3)Полностью из своих джарников, кроме Writera, который взят из libs\\ViktorComponents
    4)Полностью из своих джарников, кроме парочки Executoroв (один из которых кодирует, а другой сразу декодирует),
    которые взяты из libs\\ViktorComponents

Проверка биекции моего Executor'а выполняется заменой значения DECODE в конфиге по пути \\configs\\exec.cnfg
